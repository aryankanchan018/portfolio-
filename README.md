# portfolio-

A premium personal portfolio website for **Aryan Kanchan** — Computer Science Engineering Student, AI Developer & Software Engineer.

Built with **React + TypeScript + Tailwind CSS + Framer Motion**.

## 🚀 Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## ✨ Features
- Animated hero section
- Skills, Projects, Experience, Education sections
- Live GitHub dashboard via GitHub API
- Project detail modals
- Contact form with validation
- Scroll progress indicator
- Loading screen
- Fully responsive

## 📦 Run Locally

```bash
npm install
npm run dev
```
